# 1inch swap and transfer token cross chain using hop protocol 

## 快速开始

安装相关包

```
npm install 
```

## use 1inch to swap on Polygon 

1. 连接钱包
2. 批准使用代币
3. 发送交易 

```
node swapOn1inch.js
```


## use hop protocol transfer weth from Polygon to Arbitrum 

1. 连接钱包
2. 批准使用代币
3. 发送交易

```
node bridgeOnHop.js
```
